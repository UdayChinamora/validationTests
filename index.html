This page uses JQuery and Javascript to validate the form fields. The validation process will validate each field every time the form is submitted.

The validation enforces the following rules for the ZipCode field: Required Value, Must Be Numeric and Must have 10 characters.

When checking to see if the value that was entered is Numeric we have to account for decimal values such as 12345.6789. There are two techniques in the codes for validating that the ZipCode field is only integers. One technique rounds the value to the nearest whole number and then compares the modified value to the original value. Example Math.round(inCode) == inCode. If they are the same then the value is an integer if not there are problems.

The other technique uses the Modulus operation % to see if the remainder is zero. Example inCode % 1 == 0. If the remainder is 0 the value is an integer. If not the value is not an integer and not allowed.
